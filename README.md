# Proyecto Final - Pagina Web Basica con HTML

Este repositorio contiene una pagina web sencilla hecha con HTML y CSS para el proyecto final.

## Caracteristicas

- Estructura semantica con html, head, body, title y meta.
- Titulos con h1 y h2.
- Parrafos con p.
- Enlaces con a.
- Imagen incluida con img.
- Lista con ul.
- Formulario opcional con form, label e input.

## Archivos principales

- index.html
- styles.css

## Como publicar en GitHub Pages

1. Sube estos archivos a tu repositorio en GitHub.
2. En GitHub, entra a Settings del repositorio.
3. Ve a Pages.
4. En Build and deployment, selecciona:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
5. Guarda los cambios y espera 1-2 minutos.
6. GitHub te mostrara la URL publica.
